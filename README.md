# FilmScale

FilmScale is a movie discovery website for browsing titles, checking ratings, reading reviews, watching trailers, and building personal watchlists.

## APIs Used

| API | Purpose |
| --- | --- |
| TMDB | Movie metadata, posters, cast, reviews, search, trending, and discovery |
| OMDb | IMDb ratings, Rotten Tomatoes scores, and Metacritic scores |

## Deployment

This project is Vercel-ready as a standard Next.js application.
